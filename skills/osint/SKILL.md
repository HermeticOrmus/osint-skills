# OSINT

## Five waves

1. **Surface map** — public identifiers, primary presence
2. **Context** — affiliations, timeline, geography
3. **Patterns** — recurring themes + anomalies
4. **Predictions** — likely next moves + watch signals
5. **Briefing** — deliverable: exec summary + findings + diagrams + sources

## Scope

Public sources only. Citation required for every claim. Stamp source dates.

## Anti-patterns

Confirmation bias, source laundering, currency confusion, single-signal extrapolation, going past public.
